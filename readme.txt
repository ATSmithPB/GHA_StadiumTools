GHA_StadiumTools contains source code for a StadiumTools Grasshopper Assembly (StadiumTools.gha) 
...for use in Rhinoceros3D + Grasshopper

StadiumTools contains source code for a StadiumTools Dynamic Link Library (StadiumTools.dll) 
...for use within .NET Framework 4.8+

StadiumTools_IO contains source code for a StadiumTools_IO Dynamic Link Library (StadiumTools_IO.dll) 
...for converting between rhinoCommon and StadiumTools objects.
